###What it does
This component takes a VCF as input and outputs a MAF

#### Requirements:

python and java
(snpeff and vcf2maf are included in the seed)


###Inputs
  * path to input VCF
  * path to output file
  * tumor id
  * normal_id (optional)


###Outputs
  * MAF file
